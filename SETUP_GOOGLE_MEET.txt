## Setup (2 minutes)

1. **Get Google Credentials**
   - Go to: https://console.cloud.google.com/
   - Create project, enable Calendar API
   - Create OAuth 2.0 desktop credentials
   - Get refresh token

2. **Add to .env**
   ```env
   GOOGLE_CLIENT_ID=your-id
   GOOGLE_CLIENT_SECRET=your-secret
   GOOGLE_REFRESH_TOKEN=your-token
   ```

3. **Restart Backend**
   ```bash
   # Stop with Ctrl+C
   # Restart: python -m uvicorn app.main:app --reload
   ```

When you schedule a consultation:
1. Google Calendar API creates real event
2. Google Meet room automatically generated
3. Calendar invites sent to both parties
4. Real meeting link returned
5. Meeting can be recorded & transcribed
